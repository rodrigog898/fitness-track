📚 Instrucciones de Configuración

🔑 Crear Personal Access Token:

1. Ve a GitHub Settings → Tokens

2. Clic en "Generate new token (classic)"

3. Selecciona el scope repo (acceso completo a repositorios)

4. Copia el token generado

📁 Configurar Repositorio:

1. Crea un repositorio en GitHub (puede ser privado)

2. Sube este archivo index.html

3. Activa GitHub Pages si quieres acceso web

4. Usa el nombre exacto del repositorio en la configuración

🚀 ¡Listo!

Una vez configurado, todos los cambios se guardarán automáticamente en tu repositorio.

No necesitas hacer nada más, solo usar la aplicación normalmente.
